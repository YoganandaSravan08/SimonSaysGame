An interactive **memory-based browser game** built using **HTML, CSS, and JavaScript**.  
The game generates a sequence of colors that the player must remember and repeat in the correct order. As the level increases, the sequence becomes longer, making the game more challenging.

---

## 🚀 Features

- Interactive color buttons with flash effects  
- Random color sequence generation  
- Increasing difficulty with each level  
- Game over detection with score display  
- Keyboard event to start or restart the game  
- Smooth UI interactions using JavaScript  

---

## 🛠️ Technologies Used

- **HTML** – Structure of the game  
- **CSS** – Styling and button animations  
- **JavaScript** – Game logic, DOM manipulation, and event handling  

---

## 🎯 How to Play

1. Press **any key** to start the game.  
2. Watch the **color that flashes**.  
3. Click the **same color button**.  
4. Repeat the **entire sequence correctly**.  
5. Each level adds a **new color to the sequence**.  
6. If you press the wrong color, the **game ends** and your score is displayed.
